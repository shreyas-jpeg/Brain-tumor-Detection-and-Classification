Brain Tumor Classification using Machine Learning

Application of automated classification techniques using Machine Learning(ML) and Artificial Intelligence(AI)has consistently shown higher accuracy than manual classification.

In this web application, build a system performing detection and classification by using Deep Learning Algorithms using ConvolutionNeural Network (CNN).

This web application uses InceptionResNet model which is trained on Brain Tumor MRIs scan using transfer learning approach.

Inception-ResNet-v2 is a convolutional neural architecture that builds on the Inception family of architectures but incorporates residual connections (replacing the filter concatenation stage of the Inception architecture).

  • Upload image & Run Prediction
  • Run Machine Learning Prediction on Images
